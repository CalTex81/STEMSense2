Troubleshooting Guide

  • Server won't start: Check if port 8000 is already in use (netstat -ano | findstr :8000)
  • Backend found but offline: Check Python venv is activated
  • CORS errors: Make sure uvicorn is installed (pip install uvicorn)
  • Still can't connect: Try running on different port manually:
    uvicorn backend.server:app --port 8001
